Встраивание в Genially

✅ Вставка кодом — без HTML-файла и без GitHub. С включённым автозапуском кнопка «Старт» не нужна, поэтому ничего не «зависает».
  1. Нажмите «СОЗДАТЬ» — код появится в поле ниже.
  2. Нажмите «СКОПИРОВАТЬ».
  3. В Genially: Вставка → Виджет / Вставить код (Insert → Insert code / Embed) → вставьте код и подтвердите.
  4. В редакторе Genially iframe может не кликаться — проверяйте через «Просмотр» или после публикации.
Для Genially оставьте srcdoc. Если конкретный блок Genially не примет код — переключите на Data-URI. Кнопка «↓ HTML» внизу — запасной вариант (скачать файл и при желании выложить ссылкой).
